## Free プラン

個人開発者やお試しに最適です。

- リポジトリ 1 つまで
- 月間 20 件まで Issue を処理
- GitHub Copilot エージェントのみ
- クレジットカード不要

## Pro プラン

本格的な開発チームの生産性を最大化します。

- リポジトリ 5 つまで
- Issue 処理数は無制限
- 全 AI エージェント（Copilot、Claude、Codex）に対応
- パイプライン監視 & 自動修復
- SRE エージェント
- AutoPilot モード
- AI バックログ生成
- Agentic Chat
- ワークフローエディタ
- セルフホスト Runner

## Team プラン

Organization 全体での導入に最適なエンタープライズグレードのプランです。

- リポジトリ数無制限
- マルチメンバー管理
- 優先サポート
- 99.9% SLA
- 監査ログ
- SSO / SAML 対応
- カスタムルーティングルール
- 専用サポートチャンネル

## 請求

### 支払い方法
- クレジットカード（Visa、Mastercard、AMEX）
- Stripe による安全な決済

### 請求サイクル
- 月次請求
- 各月の月初に課金

### プラン変更
- ダッシュボードの **Settings** → **Billing** からいつでも変更可能
- アップグレード：即時反映
- ダウングレード：次の請求サイクルの開始時に反映

### 解約
- いつでも解約可能
- 現在の請求期間の終了までアクセス継続
- 解約後 30 日間はデータを保持

## よくある質問

### Q: API キーの費用は別途かかりますか？
**A:** はい。Claude（Anthropic）や Codex（OpenAI）を使用する場合、それぞれの API 使用料が別途かかります。GitHub Copilot は GitHub サブスクリプションに含まれています。

### Q: Free プランから始めて後からアップグレードできますか？
**A:** はい。Free プランから始めて、いつでも Pro や Team にアップグレードできます。

### Q: Team プランのシート数に制限はありますか？
**A:** ありません。必要な分だけシートを購入できます。

### Q: 年払いプランはありますか？
**A:** 近日対応予定です。年払いで約 20% 割引を予定しています。
