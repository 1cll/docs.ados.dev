# Agentic Chat

Agentic Chat позволяет в реальном времени общаться с ИИ, который понимает контекст вашего репозитория.

## Обзор

В отличие от обычного ИИ-чата, Agentic Chat может:

- Понимать структуру кода репозитория
- Читать и редактировать файлы
- Выполнять команды терминала
- Превращать изменения в PR

## Использование

### Из панели управления

1. Выберите целевой репозиторий
2. Нажмите на иконку чата
3. Введите запрос на естественном языке

### Из CLI

```bash
# Запуск в режиме Agentic Chat
export ADOS_MODE=chat
ados start
```

## Примеры диалогов

### Вопросы о коде

```
Пользователь: Как работает аутентификация в этом репозитории?
ИИ: В проекте используется JWT-аутентификация.
    Ключевые файлы: ...
```

### Запросы на изменение кода

```
Пользователь: Добавь ограничение частоты запросов к API логина
ИИ: Понял. Вношу следующие изменения:
    1. Добавляю пакет rate-limiter
    2. Настраиваю middleware для /api/login
    3. Добавляю тесты
    
    Нажмите [Create PR] для создания Pull Request.
```

### Помощь в отладке

```
Пользователь: POST /api/orders возвращает ошибку 500. Можешь разобраться?
ИИ: Проверил orders.controller.ts.
    На L45 отсутствует обработка случая, когда items не определён.
    Вот предлагаемое исправление...
```

## Режимы

Agentic Chat поддерживает несколько режимов работы:

| Режим | Переменная окружения | Описание |
|-------|---------------------|----------|
| `chat` | `ADOS_MODE=chat` | Только чат |
| `issue` | `ADOS_MODE=issue` | Только обработка задач |
| `all` | `ADOS_MODE=all` | Чат + обработка задач |

## Безопасность

- Содержимое чата не сохраняется на сервере
- Доступ к репозиторию основан на правах пользователя
- Инструкции для ИИ управляются через `copilot-instructions.md`

> [!TIP]
> Agentic Chat идеален, когда нужна помощь ИИ без создания полноценной задачи — для быстрых вопросов или мелких исправлений.
