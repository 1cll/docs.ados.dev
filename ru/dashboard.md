# Обзор панели управления

Панель управления ADOS — это веб-консоль для управления всеми функциями.

## Страницы

| Страница | Описание |
|----------|----------|
| **Dashboard** | Статистика задач, показатели успешности, графики трендов, карточки репозиториев |
| **Pull Requests** | Список PR, созданных ИИ, статус, действия по слиянию |
| **Monitor** | Статус GitHub Actions, оповещения безопасности |
| **Runners** | Управление Self-Hosted Runner (Issue / Chat / Pipeline) |
| **Usage** | Использование токенов ИИ, оценка затрат |
| **Feedback** | Обратная связь по качеству ИИ-агентов (анализ слияний/отклонений PR) |
| **Logs** | Просмотр, поиск и фильтрация логов |
| **Settings** | Настройки репозиториев, интеграция с GitHub, уведомления, биллинг |

## Страница Dashboard

Главная панель отображает следующую информацию:

- **Карточки репозиториев** — Статус задач по каждому репозиторию (Open / Queued / Needs Attention)
- **Статистика задач** — Количество обработанных и процент успеха
- **Графики трендов** — Тренды обработки за день/неделю
- **Производительность агентов** — Сравнение Copilot / Claude / Codex

## Страница Pull Requests

Список Pull Request'ов, созданных ИИ.

- **Статус**: Open / Merged / Closed
- **Быстрое слияние**: Слияние или squash прямо из панели
- **Предпросмотр изменений**: Сводка изменений

## Страница Usage

Визуализация использования токенов ИИ-агентов и затрат.

- **График ежедневного использования**
- **Разбивка по агентам**
- **Настройка бюджета** — Установка месячного лимита и оповещения при превышении

## Страница Settings

### Настройки репозиториев

- Добавление, редактирование и удаление репозиториев
- Настройка меток, целевых веток и агентов по умолчанию
- Правила маршрутизации задач

### Интеграция с GitHub

- Установка и управление GitHub App
- Регистрация Personal Access Token (PAT)
- Проверка статуса подключения

### Настройки уведомлений

- Настройка Slack / Discord / Email
- Правила уведомлений (фильтрация по серьёзности)
- Отправка тестовых уведомлений

### Управление биллингом

- Просмотр текущего плана
- Обновление / понижение плана
- Просмотр истории платежей

> [!TIP]
> Панель управления доступна по адресу [https://ados-platform-dashboard.web.app](https://ados-platform-dashboard.web.app).
